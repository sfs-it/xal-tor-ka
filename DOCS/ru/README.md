**Единственный охраняемый вход перед всеми вашими онлайн-сервисами.** Xal-Tor-Ka —
это «цифровой швейцар» вашей инфраструктуры: он ставит единственную укреплённую
дверь перед каждым сайтом и приложением, которые вы публикуете, и решает — для
каждого из них — кому разрешён вход. Больше никаких паролей, разбросанных по
отдельным сервисам: правило живёт в одном месте.

Для **лица, принимающего решения**, ценность проста: вы публикуете внутреннее
приложение, не открывая его всему миру; вы выбираете, будет ли сервис **открыт
для всех**, **зарезервирован для вошедших пользователей** (с двухэтапной
проверкой) или **доступен только явно авторизованным людям**; и всем этим вы
управляете из **защищённой веб-панели**. Пользователи могут входить по внутренним
учётным данным либо через корпоративную учётную запись **Google** или
**Microsoft**. Система ведёт журнал доступа, защищает себя от попыток подбора
паролей и создаёт резервные копии собственной конфигурации.

## Для кого это

- Вы держите несколько веб-сервисов (интранет, бизнес-приложения, панели,
  внутренние инструменты) и хотите **единую контролируемую точку входа** вместо
  защиты каждого по отдельности.
- Вы хотите **централизованно решать**, кто что видит, со вторым фактором
  безопасности и/или корпоративным входом (Google/Microsoft), **не изменяя сами
  сервисы**.
- Вы хотите, чтобы единственным, что открыто в интернет, был **щит**, а реальные
  сервисы оставались скрытыми за ним.

## Что вы получаете

- **Единственная укреплённая дверь**: остальная часть инфраструктуры недоступна
  напрямую из интернета.
- **Три уровня доступа** на каждый сервис: публичный · только вошедшие
  пользователи (2FA) · только люди из белого списка.
- **Вход через Google / Microsoft** или по внутренним учётным данным.
- **Защищённая веб-панель** для управления сервисами, пользователями и правами.
- **Эксплуатационная безопасность**: журнал доступа (подключаемый к системам
  предотвращения вторжений), автоматические резервные копии конфигурации,
  мониторинг состояния сервисов.
